# Kemari Website UI Portfolio

Independent public deployment package for the current Kemari Blakemore Website UI Designer portfolio.

The site root is a neutral contact page. It intentionally does not list or link to any cases.

Each industry direction is shared only with a matching prospect:

- `/nadwa/` for high-end Arabic tableware and gifting.
- `/nocturne/` for fine fragrance discovery and refill.
- `/roastery/` for specialty coffee, subscription, and wholesale.
- `/majlis/` for restaurant reservations and private dining.

The sample UI audit lives at `/ui-audit/`.

This package intentionally excludes the legacy `kemari-sample-audit` project and its older root audit page.
